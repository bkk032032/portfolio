# 포트폴리오 웹사이트

HTML, CSS, JavaScript로 만든 모던한 포트폴리오 웹사이트입니다.

## 특징

- ✨ 모던하고 반응형 디자인
- 🎨 부드러운 애니메이션 효과
- 📱 모바일 친화적 레이아웃
- 🚀 빠른 로딩 속도
- 🎯 단일 페이지 애플리케이션 (SPA)

## 섹션 구성

1. **Hero** - 메인 소개 섹션
2. **About** - 자기소개
3. **Skills** - 기술 스택
4. **Projects** - 프로젝트 포트폴리오
5. **Contact** - 연락처 및 문의 폼

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 또는 로컬 서버 실행:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (http-server 필요)
   npx http-server
   ```

## 커스터마이징

### 개인 정보 수정
- `index.html`에서 이름, 소개, 프로젝트 정보 수정
- `css/style.css`의 `:root` 변수로 색상 테마 변경
- 연락처 정보 업데이트

### 프로젝트 추가
- `index.html`의 Projects 섹션에 새로운 `.project-card` 추가
- 이미지, 설명, 기술 스택, 링크 수정

### 스킬 추가/수정
- `index.html`의 Skills 섹션에 `.skill-card` 추가/수정
- Font Awesome 아이콘 변경 가능

## 파일 구조

```
portfolio/

```
